# 🎯 PROMPT PARA JULES - MVP COMPLETO

## INSTRUÇÕES INICIAIS

Jules, preciso que você crie um MVP completo para automatizar a atualização de carteira imobiliária via WhatsApp. O prazo é 2-3 dias e o custo de infraestrutura deve ser ZERO.

---

## 📋 ESCOPO COMPLETO

### Problema
Minha esposa trabalha em uma imobiliária e hoje faz manualmente:
1. Abre planilha de imóveis (CSV/XLSX)
2. Envia WhatsApp para cada proprietário perguntando disponibilidade
3. Interpreta respostas ambíguas
4. Atualiza planilha manualmente

### Solução
Automatizar passos 2, 3 e 4 com formulário estruturado e integração com WhatsApp.

---

## 🏗️ ARQUITETURA TÉCNICA

### Stack Obrigatória (TUDO GRATUITO)
- **Frontend**: HTML + CSS + JS Vanilla (hospedado na Vercel ou GitHub Pages)
- **Backend**: Google Apps Script (serverless, gratuito)
- **Banco Dados**: Google Sheets (gratuito, visual)
- **WhatsApp**: Meta Cloud API (1000 conversas/mês gratuitas)

3. Google Apps Script (Code.gs)
Endpoints:

GET - Buscar dados do imóvel

POST - Enviar mensagem WhatsApp
javascript
POST /
Payload: {
  action: "sendMessage",
  telefone: "+5511999999999",
  nome: "João Silva",
  endereco: "Rua Congonhas do Norte, 144",
  id_imovel: "001",
  form_url: "https://.../form.html?id=001"
}
POST - Salvar resposta
javascript
POST /
Payload: {
  action: "saveResponse",
  id_imovel: "001",
  status: "disponivel",
  novo_preco: 250000,
  data_resposta: "2026-07-02T17:30:00Z"
}
Configurações:

javascript
const CONFIG = {
    SPREADSHEET_ID: 'ID_DA_PLANILHA',

Tratamento de Erros:

Log detalhado para debug

Respostas padronizadas: { success: boolean, error?: string }

# 🏠 Carteira Imobiliária - Atualização via WhatsApp

[![Status](https://img.shields.io/badge/Status-Em%20Desenvolvimento-yellow)]()
[![License](https://img.shields.io/badge/License-MIT-blue.svg)]()
[![WhatsApp](https://img.shields.io/badge/WhatsApp-API-25D366)]()

Sistema MVP para automatizar a atualização de disponibilidade de imóveis via WhatsApp com **custo zero** de infraestrutura.

## 📋 Visão Geral

Minha esposa trabalha em uma imobiliária e hoje faz manualmente:
1. Abre a planilha de imóveis (CSV/XLSX)
2. Envia WhatsApp para cada proprietário
3. Interpreta respostas ambíguas
4. Atualiza a planilha manualmente

**Este sistema automatiza os passos 2, 3 e 4!**

## 🚀 Tecnologias

| Tecnologia | Uso | Custo |
|------------|-----|-------|
| **HTML + CSS + JS** | Frontend | ✅ R$ 0 |

| **Google Sheets** | Banco de Dados | ✅ R$ 0 |
